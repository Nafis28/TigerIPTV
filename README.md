
# Tiger IPTV Player

Tiger IPTV is a Free IPTV player for Android devices. It lets you add your IPTV playlist, browse channels, search quickly, save favourites, and watch live TV in fullscreen.

## Download and Install

Download the APK file from this GitHub page.

The file will look something like:
https://github.com/Nafis28/TigerIPTV

To install it:

1. Download the APK to your Android phone, tablet, Android TV, or TV box.
2. Open the APK file.
3. If Android asks for permission, allow installation from unknown apps.
4. Tap **Install**.
5. Open **Tiger IPTV** from your apps list.

## Important

Tiger IPTV does **not** include TV channels, movies, playlists, or subscriptions.

You need to add your own IPTV M3U playlist URL from your IPTV provider.

## Features

* Add your own M3U IPTV playlist URL.
* Save more than one playlist.
* Browse live TV channels in a clean list.
* Search for channels by name.
* Organise channels by groups when the playlist supports it.
* Mark channels as favourites.
* See recently watched channels.
* Watch channels in fullscreen.
* Change the video view between Fit, Zoom, and Fill.
* Choose stream quality when different options are available.
* Automatically remembers your playlists, favourites, and recent channels.
* Shows channel logos when provided by the playlist.
* Reloads channels from cache if the playlist cannot refresh.
* Includes a simple retry option if a stream fails to play.
* Works well on Android TV and TV boxes.


## How to Use

1. Open **Tiger IPTV**.
2. Tap **Playlist**.
3. Enter a playlist name.
4. Paste your M3U playlist URL.
5. Tap **Save**.
6. Select a channel and start watching.

To favourite a channel, long-press it.

## Notes

Some channels may not play if the stream is offline, blocked, expired, or not supported by your device.

If your playlist does not load, check that the M3U URL is correct and active.
